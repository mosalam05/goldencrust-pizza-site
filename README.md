A simple responsive website for GoldenCrust Pizza Restaurant showcasing menu, special offers, statistics, and contact information.

## Features

- **Home Section:** Eye-catching hero with pizza image and call-to-action button.
- **Special Offers:** Highlighted deals with images and prices.
- **Menu Section:** Grid layout for pizza options with sizes and prices.
- **Statistics Section:** Showcases restaurant stats like customer satisfaction and dishes served.
- **Footer:** Contact info, opening hours, and social media links.

## Technologies Used

- HTML5
- CSS3 (with responsive design)
